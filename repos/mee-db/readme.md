# Mee DB

## Overview
This project is designed to work with binary storage using specific query language focused on a user. It provides results in json format.

## Rest API to play with mock store (http://localhost:8080/swagger-ui/):
```bash
cd mee_ql_api
cargo run
```

## Build tree-sitter lib
```bash
cd tree_sitter_mee_ql
tree_sitter generate --build
```
